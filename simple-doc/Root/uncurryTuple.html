uncurryTuple ()

  • uncurryTuple<A, B, C>(_): ((_) => C)

  • Transforms the given function of two arguments from curried format to "tuple curried" which can be used with Opt.zip.

    const sub = (x: number) => (y: number) => x - y;
    opt(4) // Some(4)
      .zip(opt(1)) // Some([4, 1])
      .map(uncurryTuple(sub)) // Some(3)
