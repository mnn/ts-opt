isEmpty ()

  • isEmpty(x): boolean

  • Similar to isEmpty from lodash, but also supports Opts. Returns true for None, [], null, undefined, empty map, empty set, empty object, '' and NaN. Otherwise returns false. * *

    Parameters

    • x: PossiblyEmpty

    Returns boolean

    Example

    •

    isEmpty(opt(1)) // false isEmpty(opt(null)) // true isEmpty([]) // true isEmpty([1]) // false isEmpty(null) // true isEmpty('') // true
