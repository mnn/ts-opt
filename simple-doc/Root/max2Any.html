max2Any ()

  • max2Any(a): ((b) => Opt<number>)

  • Get a larger number from two possibly empty numbers. Returns None when both operands are None, otherwise returns the other (nonempty) operand.

    Parameters

    Returns ((b) => Opt<number>)

    Example
