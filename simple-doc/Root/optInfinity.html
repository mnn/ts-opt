optInfinity ()

  • optInfinity(x): OptSafe<number>

  • For numbers equal to Infinity or -Infinity returns None, otherwise acts same as opt.

    optInfinity(undefined) // None
    optInfinity(1) // Some(1)
    optInfinity(0) // Some(0)
    optInfinity(Infinity) // None
    optInfinity(-Infinity) // None

    Parameters

    • x: undefined | null | number

    Returns OptSafe<number>
