deserializeUnsafe ()

  • deserializeUnsafe(x): Opt<unknown>

  • Unsafe version of deserializeOrCrash. Deserialization failure is indistinguishable from deserialized None. It is usually better to use deserialize or deserializeOrCrash.

    Parameters

    • x: unknown

      input value to be deserialized

    Returns Opt<unknown>

    deserialized value as an Opt instance

    Example

    deserializeUnsafe({ type: 'Opt/None' }) // none
    deserializeUnsafe({ type: 'Opt/Some', value: 0 }) // some(0)
    deserializeUnsafe(9) // none
