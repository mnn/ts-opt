contains (method)

Abstract contains

  • contains(x): boolean

  • Compares inner value with given value using ===. Always false for None.

    some(1).contains(1) // true
    some(0).contains(1) // false
    none.contains(undefined) // false

    Similar to JavaScript Array#includes method.

    Parameters

    • x: T

    Returns boolean
