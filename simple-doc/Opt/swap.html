swap (method)

Abstract swap

  • swap<U>(newValue): Opt<U>

  • Swaps value inside (for None it's noop).

    Type Parameters

    • U

    Parameters

    • newValue: U

    Returns Opt<U>

    Example

    opt(1).swap('a') // Some('a')
    none.swap('a') // None

    Same as map(const(newValue)).
