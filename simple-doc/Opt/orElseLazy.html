orElseLazy (method)

Abstract orElseLazy

  • orElseLazy(def): T

  • Get inner value of Some, or lazily use passed def value for None.

    If a computation of a default value is not expensive (or has been already computed), use orElse instead.

    Parameters

    • def: (() => T)
        • (): T

        • Returns T

    Returns T

    Example

    some(1).orElseLazy(() => 2) // 1
    none.orElseLazy(() => 2) // 2
