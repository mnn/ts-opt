lengthIn (method)

lengthIn

  • lengthIn<R>(): R

  • Returns the length of a string or array wrapped in an Opt.

    Type Parameters

    • R extends Opt<number>

    Returns R

    Example

    opt('hello').lengthIn() // Some(5)
    opt([1, 2, 3]).lengthIn() // Some(3)
    opt('').lengthIn() // Some(0)
    opt([]).lengthIn() // Some(0)
    none.lengthIn() // None
