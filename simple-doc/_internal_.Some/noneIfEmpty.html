noneIfEmpty (method)

noneIfEmpty

  • noneIfEmpty(this): Opt<Exclude<T, "" | [] | EmptyValue | None<any>>>

  • Returns None if opt holds a value for which isEmpty returns true, otherwise passes opt unchanged.

    Parameters

    • this: Opt<PossiblyEmpty>

    Returns Opt<Exclude<T, "" | [] | EmptyValue | None<any>>>

    Example

    opt('x').noneIfEmpty() // Some('x')
    opt('').noneIfEmpty() // None
    opt([]).noneIfEmpty() // None
    opt({}).noneIfEmpty() // None
